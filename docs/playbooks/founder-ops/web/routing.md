# Web Operator Routing Note

Use `web` as the owner lane when the request is primarily about business web surfaces and CTA operations.

## Route to `web` for
- website copy/structure
- landing changes
- CTA optimization
- proof pages
- web maintenance

## Internal routing modes

- `PAGE_PACKET`
- `CTA_REVIEW`
- `SURFACE_AUDIT`
- `PROOF_PAGE`
- `MAINTENANCE`
- `READINESS_SUMMARY`
- `ITERATION`

## Standard delegation packet

Every Jarvis handoff should include:
- Objective
- Context
- Constraints
- Requested output
- Verification requirement
- Urgency
- Stop condition
