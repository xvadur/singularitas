# Runtime Outputs

This tree is for runtime evidence, runtime handoffs, and reviewable runtime artifacts.

- `runtime-evidence/` for verification traces, smoke tests, and runtime proof
- `runtime-handoffs/` for transition material between agents or systems

Business-facing canonical deliverables belong in `firma`.
Legacy output subtrees were moved into `archive/cutover-2026-03-18/legacy-outputs/` during the workspace cleanup.

Use `outputs/` for:
- runtime handoffs
- validation evidence
- reviewable traces
- internal runtime artifacts that still belong inside the active runtime
