# Source of Truth 2026

## Canonical surfaces

- tasks -> `control/tasks/`
- habits -> `control/habits/`
- roadmaps -> `control/roadmaps/`
- founder cockpit -> `control/cockpit/`
- daily founder control -> `control/daily/`
- personal mail state -> `personal/mail/`
- personal commitments -> `personal/commitments/`
- CRM and pipeline -> `business/crm/`
- revenue execution -> `business/revenue/`
- web and funnel state -> `business/web/`
- content operations -> `business/content/`
- voice factory -> `business/voice/`
- automation operations -> `business/automation/`
- initiative execution -> `projects/`
- raw sync data -> `data/`
- reviewable exports -> `outputs/`

## External roles

- `singularitas_opus` -> raw capture, journaling, scratch thinking
- `firma` -> legacy import, archive reference, historical business material

## Startup contract

Jarvis startup should read:

1. root bootstrap files
2. `memory/` today and yesterday
3. `control/cockpit/status.md`
4. today's `control/daily/YYYY-MM-DD/brief.md`
5. relevant surfaces under `control/`, `personal/`, `business/`, and `projects/`
6. `singularitas_opus` only if raw founder context is needed
