# Shared Agent Runtime Docs

This folder holds shared doctrine for the multi-agent system.

- `rules/` for global agent rules and architecture
- `briefing/` for shared context that multiple agents should inherit
- `contracts/` for runtime contracts and role definitions
- `initializers/` for reusable boot or initialization material

Only keep shared runtime truth here. If a note is business strategy rather than agent doctrine, move it to `business/`.
