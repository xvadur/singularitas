# Ironman quick checklists

## Sales sprint
- Pull target list
- Pick top priority contacts
- Draft/send outreach
- Log interactions
- Set reminders/follow-ups

## Build sprint
- Confirm scope
- Implement smallest valuable increment
- Verify result
- Log changelog / issue state

## Ops sprint
- Identify bottleneck
- Update source-of-truth systems
- Resolve stale reminders/tasks
- Create next checkpoint

## Content sprint
- Select topic with signal
- Draft hook + skeleton
- Get user hot-take
- Save to queue (draft/approved)
