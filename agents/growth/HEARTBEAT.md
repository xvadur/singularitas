# HEARTBEAT.md

On heartbeat:
- read the current project KPI file and watchlist
- detect drops, opportunities, stale offers, and missing next actions
- prepare copy diffs, outreach drafts, CRM tasks, or funnel recommendations
- if nothing material changed, reply `HEARTBEAT_OK`

Do not:
- send outbound communication automatically by default
- rewrite live assets without a reviewable patch or handoff
