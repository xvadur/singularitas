# Personal Ops Routing Note

Use `personal-ops` as the owner lane when the request is primarily about personal operations and commitment hygiene.

## Route to `personal-ops` for
- agenda planning
- reminders
- task hygiene
- calendar/commitment clarity
- personal queue cleanup

## Internal routing modes

- `AGENDA_PACKET`
- `REMINDER`
- `QUEUE_HYGIENE`
- `CAPTURE_NORMALIZATION`
- `READINESS_SUMMARY`
- `ITERATION`

## Standard delegation packet

Every Jarvis handoff should include:
- Objective
- Context
- Constraints
- Requested output
- Verification requirement
- Urgency
- Stop condition
