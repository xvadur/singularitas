# Personal Web Routing Note

Use `personal-web` as the owner lane when the request is primarily about personal web surfaces and maintenance.

## Route to `personal-web` for
- personal site updates
- maintenance
- copy refresh
- personal surface audit

## Internal routing modes

- `SURFACE_AUDIT`
- `CHANGE_PACKET`
- `MAINTENANCE`
- `COPY_REFRESH`
- `READINESS_SUMMARY`
- `ITERATION`

## Standard delegation packet

Every Jarvis handoff should include:
- Objective
- Context
- Constraints
- Requested output
- Verification requirement
- Urgency
- Stop condition

- `templates/delegation-packet-example.md`
