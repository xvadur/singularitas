# Runtime Mission Template

Copy this folder when creating a new runtime mission home.

Minimum required files:

- `brief.md`
- `status.md`
- `metrics.md`
- `watchlist.md`

Required subfolders:

- `approvals/`
- `spec/`
- `artifacts/`
- `validation/`
- `content/`
- `staging/`
- `handoffs/`
- `approved-for-firma/`

Every mission home must include these header fields in `brief.md`:

- `canonical_owner: singularitas`
- `firma_record: /absolute/path/in/firma`
- `status_scope: runtime only`
- `do_not_store_business_truth_here: true`
