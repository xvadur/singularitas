# Editorial Brainstorm

Single source for editorial pre-writing.

## Modes

- `quick` (default): fast social commentary prep
- `deep`: longer-form post planning (blog/newsletter)

## Quick mode (social post)

1. Generate 3 POV angles.
2. Recommend one angle + why.
3. Produce:
- 1 hook
- 3-point argument skeleton
- 1 CTA
4. Ask for approval before full draft.

## Deep mode (blog/long-form)

1. Define target reader and intended business outcome.
2. Define thesis (1 sentence) and anti-thesis.
3. Propose outline:
- opening
- 3 core sections
- closing + CTA
4. Add risk check (hot takes, legal/reputation sensitivity).
5. Ask for brief approval.

## Rules

- No generic filler.
- No full draft before brief approval.
- Tie output to channel goal (engagement, authority, lead, conversion).
