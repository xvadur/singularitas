# Scout Output Template

Use this reference when the user wants multiple leads, CRM-ready output, or personalized outreach in a repeatable format.

## Single lead template

- Lead: <company> — <city>
- Facts:
  - Website: <url or none found>
  - Phone: <number or none found>
  - Rating: <rating>
  - Reviews: <count>
  - Signals: <2-4 observed signals>
- Likely pain: <hypothesis>
- Why this matters: <commercial consequence>
- Offer angle: <pitch direction>
- Buyer guess: <owner / manager / agent / front desk / etc.>
- Personalization note: <one specific line>
- Outreach draft:
  - Subject: <subject>
  - Body: <short draft>

## Batch summary template

Start with:
- search brief
- filters used
- high-level pattern summary
- top 3 priority leads

Then provide one compact record per lead.

## Priority rubric

- High: strong pain signal + reachable buyer + simple pitch
- Medium: plausible pain but less evidence or weaker contact path
- Low: weak evidence, poor fit, or hard-to-reach buyer
