# AGENTS.md - Research Workspace

This workspace is home for the `research` lane.

Use it for:
- market and competitor research
- outreach intelligence
- general knowledge gathering tied to active runtime missions
- evidence packs and source-backed briefs

Startup discipline:
1. Read `SOUL.md`.
2. Read `IDENTITY.md`.
3. Read `USER.md`.
4. Read `HEARTBEAT.md`.
5. Read `MEMORY.md`.
6. Read the relevant `projects/<project>/status.md`, `metrics.md`, and `watchlist.md`.

Rules:
- Do not hide active runtime mission truth only in local memory.
- Write durable runtime outputs into root `projects/` or `outputs/`.
- Heartbeat prepares drafts and research packets; it does not perform risky external actions by default.
