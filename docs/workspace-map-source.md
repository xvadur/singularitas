# Mapa vaultu

## Root priečinky

- `+` = capture
- `Kontext` = tematický knowledge graph
- `Operácie` = execution podľa iniciatív
- `Calendar` = plugin-managed daily a periodic notes
- `Denník` = denná a časová vrstva
- `Archív` = neaktívne veci
- `Systém` = pravidlá a navigácia

## Krátky model

Capture -> Kontext -> Operácie -> Archív

`Calendar` je technická výnimka kvôli plugin workflow pre daily a periodic notes.

Nie každá poznámka musí prejsť všetkými vrstvami.
