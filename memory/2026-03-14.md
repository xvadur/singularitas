## Legacy task-layer findings

- The experimental stack was technically reachable locally, but permissions/orchestration quality were weak.
- A key distinction surfaced: OpenClaw gateway scopes do not automatically grant business-level task permissions in external task systems.
- Unassigned task creation worked, but delegated execution was blocked until higher permissions were granted.
- At the time of audit, the operator path was healthy while some worker agents were in `error` state, so part of the problem was runtime failure rather than auth alone.
